What is a Data Source?

A data source represents an external entity that provides water quality data to your system. Each import is associated with a data source to track data provenance and maintain audit trails.

Custom Parsers:

If your data source uses a non-standard format, you can create a custom parser. Contact your system administrator for assistance with implementing custom parsers.
